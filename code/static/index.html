راهنمای استفاده

۱. برای ضبط صدا، روی "شروع ضبط" کلیک کنید و اجازه دسترسی به میکروفون را بدهید.

۲. پس از ضبط، روی "توقف ضبط" کلیک کنید تا متن تبدیل شده نمایش داده شود.

۳. برای آپلود فایل صوتی (WAV، MP3 یا WebM)، فایل را انتخاب کرده و روی "آپلود و تبدیل فایل" کلیک کنید.

۴. متن تبدیل شده نمایش داده شده و قابل دانلود است.

نکته: برای ضبط صدا، باید از طریق HTTPS به برنامه دسترسی داشته باشید.
